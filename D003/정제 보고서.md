## 2. 진단 결과

- 의심되는 결측 유형: amount는 app 채널에 쏠림 → MAR

## 3. 처리 결정과 근거
| 이슈 | 결정 | 근거 | 한계 |
| --- | --- | --- | --- |
| amount 결측 | 채널별 중앙값 대체 | app 결측 비율 높음, MAR 가설 부합 | 채널 외 다른 요인일 가능성 존재 |
| price 결측 | 카테고리별 중앙값 대체 | 결측 비율 적음, 카테고리별 가격대 보존 | 표본이 적은 카테고리는 통계 불안정 |
| customer_age = 999, 0 | NaN 처리 후, 중앙값 대체 | 물리적으로 불가능한 값으로, 입력 실수 추정 | 실제 연령 정보 복원 불가능 |
| quantity = 200 | NaN 처리 후, 중앙값 대체 | 일반 소비자 도메인 외 값 | 도매 가능 고객인지 확인 필요 |
| amount = 50,000,000 | 유지 + amount_outlier 플래그 | 실제 고액 거래일 가능성 | 도매일 경우 일반 소비자와 분류해 별도 분석 필요 |
